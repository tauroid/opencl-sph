The numerical data is all stored in a contiguous array at data, to facilitate uploading to the GPU.

When data needs to be used by MATLAB, sync_to_mex should be used to copy the flat stored data to the MEX managed arrays in host_data. These are kept host-side throughout as I don't know anything about their internal structure.
